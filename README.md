# Maternal Delivery Method Prediction

A machine learning project focused on analyzing maternal health data to identify the factors that influence a mother's delivery method and predict whether the delivery will be **Natural** or **Cesarean**.

## Project Overview

This project was developed using a real-world dataset collected from pregnant mothers in a hospital. The main objective was to analyze maternal and clinical factors associated with the delivery process and build predictive models capable of classifying the expected delivery method.

The project combines **data analysis**, **feature engineering**, **data visualization**, and **machine learning** to gain insights into the factors that contribute to delivery outcomes.

## Objectives

- Analyze real hospital data collected from pregnant mothers.
- Explore the relationship between maternal and clinical characteristics and the delivery method.
- Identify the most influential factors affecting whether a delivery is **Natural** or **Cesarean**.
- Perform comprehensive feature analysis using:
  - Correlation analysis
  - Multicollinearity analysis
  - Non-linear relationship analysis between features and the target variable
  - Decision tree rule extraction to interpret influential decision patterns
  - Statistical and visual data exploration
- Build and evaluate machine learning models to predict the expected delivery method.

## Project Workflow

1. Data preprocessing and cleaning
2. Exploratory Data Analysis (EDA)
3. Correlation analysis between features
4. Multicollinearity analysis to identify redundant variables
5. Analysis of linear and non-linear relationships between each feature and the delivery method
6. Decision tree analysis to extract interpretable decision rules
7. Feature selection and engineering
8. Model training, tuning, and evaluation
9. Prediction of the delivery method (Natural or Cesarean)

## Results

The analysis provided valuable insights into the clinical and demographic factors that may influence the delivery method. Several machine learning models were trained and evaluated to predict whether a mother would have a **Natural Delivery** or a **Cesarean Section**, demonstrating the potential of data-driven approaches in healthcare decision support.
